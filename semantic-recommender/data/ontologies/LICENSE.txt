**License:** Apache 2.0 (assumed for academic research use)
**Attribution Required:** Yes

AdA Filmontology - A Common Semantic Representation for Audiovisual Annotations
Developed by: ProjectAdA team
Copyright: Project AdA contributors

Citation:
```
ProjectAdA. "AdA Film Ontology". GitHub repository.
https://github.com/ProjectAdA/public/tree/master/ontology
```

**Description:**
- 8 annotation levels, 78 types, 502 predefined values
- Film-analytical terms for fine-grained semantic video annotations
- Languages: German and English
- Focus: Camera, Editing, Sound, Lighting, Color, Acting, Mise-en-sc√®ne, Narrative

**Usage in this project:**

**Source:** https://github.com/robotenique/movies-ontology
**License:** MIT (assumed)
**Attribution Required:** Yes

Movies Ontology - General movie metadata ontology
Author: robotenique (GitHub user)
Copyright: Original author

Citation:
```
robotenique. "Movies Ontology". GitHub repository.
https://github.com/robotenique/movies-ontology
```

**Description:**
- General movie metadata ontology
- Coverage: Movies, actors, directors, genres, ratings, studios, awards
- Format: OWL + RDF

**Usage in this project:**
Used for mapping MovieLens metadata (genres, ratings, directors) to ontology classes
for hybrid semantic+metadata recommendations.

## 3. OMC (Ontology for Media Creation) - OPTIONAL

**Source:** https://github.com/MovieLabs/OMC
**License:** Apache License 2.0
**Attribution Required:** Yes

Ontology for Media Creation (OMC)
Copyright: MovieLabs (Motion Picture Laboratories, Inc.)
Licensed under: Apache License 2.0

Apache License 2.0 - Full text: http://www.apache.org/licenses/LICENSE-2.0

Citation:
```
MovieLabs. "Ontology for Media Creation (OMC)". GitHub repository.
https://github.com/MovieLabs/OMC
```

**Description:**
- Complete lifecycle of audiovisual content production
- Namespaces: cw: (Creative Works), omc: (Production), omd: (Distribution)
- Industry standards aligned with EIDR
- Format: JSON Schema + RDF/Turtle

**Usage in this project:**
Optional ontology for production-aware recommendations. Currently not used as
MovieLens dataset lacks production metadata. May be integrated in future phases.
